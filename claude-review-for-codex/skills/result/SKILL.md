# Claude Review for Codex Result

Resolve `<plugin-root>` as two directories above this `SKILL.md`.

Run:

```bash
node "<plugin-root>/scripts/claude-review-for-codex.mjs" result $ARGUMENTS
```

Arguments may be a review id or a background job id.
